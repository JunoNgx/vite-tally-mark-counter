# Tally Mark Counter

An experimental exercise in which I use Vite to attempt the [Counter-themed Codepen's June Challenge](https://codepen.io/challenges/2022/june/1).

# Tech stack
* Vite
* Svelte
* SASS
* Pug

#  How to run
* Clone the repo
* Install packages with `yarn`
* Run `yarn run dev`
